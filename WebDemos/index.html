Visual Computing Web Demos


About

Welcome to the demos of the Visual Computing Lab. The presented web applications are used within lectures to demonstrate concepts of Computer Graphics, Image Processing and Information Visualization. The demos include used JavaScript libraries in source code to improve debugging (and to run locally without Internet access). Feel free to use the source code of these demo applications as starting point for your own projects.
