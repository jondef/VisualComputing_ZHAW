Readme: Visual Computing Web Demos


About

Welcome to the Web demos of the Visual Computing Lab. The presented web applications are used within lectures to demonstrate concepts of Computer Graphics, Image Processing and Information Visualization.


Installation

All demos will run locally without the need to be online (which is not always the case being in a classroom). In order to run the demos proberly a local Web server has to be used due to security restrictions of Web browsers in accessing local files.

Mac OSX: Let's use Python that is included in OSX by default.
1. Open terminal and navigate to the "demos" folder.
2. Run python -m SimpleHTTPServer to start local Web server.
(If Python 3.X is installed use python3 -m http.server)
3. Open localhost:8000 in your preferred Web browser.

Linux: follow the Mac OSX instructions.

Windows: Install Python and follow the Mac OSX instructions.
Alternative: run full-fledged web server like Apache.


WebGL Browser Compatibility

To check if your browser runs WebGL, visit get.webgl.org.

Chrome: WebGL is enabled by default.
Chrome for Android: WebGL is enabled by default (version 36+).
Firefox: WebGL is enabled by default (version 30+).
Safari on OSX 10.9 (Maverick): Activate WebGL under "Developer" menu.
Safari on OSX 10.10 (Yosemite): WebGL is enabled by default.
Safari on iOS 7: WebGL not supported.
Safari on iOS 8: WebGL is enabled by default.
Opera: WebGL is enabled by default (version 23+).
Internet Explorer Version 11: partial support of the WebGL spec.



Library Versions

In order to run the demos offline and to improve programming/debugging the used JavaScript libraries are installed locally and non-minimized. For productive use link to the minimized library versions. Currently the following library releases are used:

SVG.js: 2.6.1
D3.js: 4.8.0
THREE.js: r85


Problems

SVG Clock is not updated to the latest version of svg.js because it's no longer actively maintained by the developer. The older versions of these files are stored exclusively for that demo, so it is still operational.
